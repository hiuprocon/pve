ProconVE
========================================

ProconVE is a toolkit for creating a 3D Virtual Environment
for programming contests.

### Build

Install Java JDK (1.6 or later) and ant (1.9?).
Then type...

    ant get-lib
    ant get-res
    ant

### Execute

Type...

    ant PServer3
